# Obsidian MCP Servers - Quick Reference\n\nThe claude-desktop-mcp-playground now supports **two** different Obsidian MCP servers. Choose the one that best fits your needs:\n\n## 🔄 Quick Comparison\n\n| Feature | `obsidian` (Direct) | `obsidian-mcp-server` (REST API) |\n|---------|-------------------|----------------------------------|\n| **Setup** | Simple | Requires plugin |\n| **Performance** | Good | Better (with caching) |\n| **Features** | Basic | Advanced |\n| **Integration** | File-only | Works with live Obsidian |\n| **Safety** | Good | Better (atomic operations) |\n\n## 🚀 Quick Install\n\n### Option 1: Direct File Access (`obsidian`)\n```bash\n# Simple setup - no Obsidian plugin needed\npg config install obsidian --arg vault_path=\"C:\\\\Users\\\\username\\\\vault\"\n```\n\n### Option 2: REST API (`obsidian-mcp-server`)\n```bash\n# Advanced setup - requires Local REST API plugin\npg config install obsidian-mcp-server --env OBSIDIAN_API_KEY=\"your-api-key\"\n```\n\n## ⚙️ Configuration Examples\n\n### Both Servers (Recommended)\n```json\n{\n  \"mcpServers\": {\n    \"obsidian\": {\n      \"command\": \"npx\",\n      \"args\": [\"-y\", \"mcp-obsidian\", \"C:\\\\Users\\\\seanp\\\\seans-vault\"],\n      \"env\": {}\n    },\n    \"obsidian-mcp-server\": {\n      \"command\": \"npx\",\n      \"args\": [\"obsidian-mcp-server\"],\n      \"env\": {\n        \"OBSIDIAN_API_KEY\": \"your-api-key-here\",\n        \"OBSIDIAN_BASE_URL\": \"https://127.0.0.1:27124\"\n      }\n    }\n  }\n}\n```\n\n## 🛠️ Setup Steps\n\n### For Direct Access (`obsidian`)\n1. Run: `pg config install obsidian --arg vault_path=\"/path/to/vault\"`\n2. Restart Claude Desktop\n3. ✅ Ready to use!\n\n### For REST API (`obsidian-mcp-server`)\n1. **Install plugin in Obsidian:**\n   - Settings → Community plugins → Browse\n   - Search \"Local REST API\" → Install & Enable\n2. **Get API key:**\n   - Plugin settings → Copy API key\n3. **Install server:**\n   ```bash\n   pg config install obsidian-mcp-server --env OBSIDIAN_API_KEY=\"paste-key-here\"\n   ```\n4. **Restart Claude Desktop**\n5. ✅ Ready with advanced features!\n\n## 🎯 When to Use Which?\n\n### Use `obsidian` (Direct) when:\n- ✅ You want simple setup\n- ✅ You don't need advanced features\n- ✅ You're just getting started\n- ✅ You don't want to install Obsidian plugins\n\n### Use `obsidian-mcp-server` (REST API) when:\n- ✅ You want the best performance\n- ✅ You need atomic operations (safer)\n- ✅ You want advanced search and filtering\n- ✅ You're using Obsidian actively\n- ✅ You need frontmatter management\n\n### Use Both when:\n- ✅ You want maximum flexibility\n- ✅ You need different tools for different tasks\n- ✅ You're developing/testing MCP integrations\n\n## 🔍 Available Tools\n\n### `obsidian` Tools\n- `obsidian_list_notes` - List vault contents\n- `obsidian_read_note` - Read note content\n- `obsidian_update_note` - Update notes\n- `obsidian_global_search` - Search vault\n- Basic tag and frontmatter operations\n\n### `obsidian-mcp-server` Tools  \n- All basic tools plus:\n- `obsidian_manage_frontmatter` - Atomic frontmatter editing\n- `obsidian_manage_tags` - Advanced tag operations\n- `obsidian_search_replace` - Search and replace\n- Enhanced search with filters and pagination\n- Directory management and caching\n\n## 🆘 Troubleshooting\n\n### \"Usage: mcp-obsidian <vault-directory>\" Error\n✅ **Fixed!** Update to latest registry - uses correct command line args now.\n\n### \"Connection refused\" for REST API\n- ✅ Check Local REST API plugin is enabled\n- ✅ Verify API key is correct\n- ✅ Check URL (HTTP: 27123, HTTPS: 27124)\n\n### Want to test?\n```bash\n# Check what's available\npg config search obsidian\n\n# Get detailed info\npg config info obsidian\npg config info obsidian-mcp-server\n```\n\nFor detailed setup guide, see [obsidian-server-update.md](obsidian-server-update.md).\n
